# EBM

A component to encapsulate the running of EBM measures (both HEDIS and non-HEDIS)

## Development Guidelines

### Branching Stragegy

This repository utilizes full [git-flow](https://datasift.github.io/gitflow/IntroducingGitFlow.html)

### Program Headers

Each program should have an appropriate header on the file using the appropriate template from [here](https://indy-github.milliman.com/PRM/analytics-pipeline/tree/develop/000_Templates/QRM_Header_Templates/).
